# 🎯 CAPTCHA Simulator

**CAPTCHA** (Completely Automated Public Turing test to tell Computers and Humans Apart) is a security measure used to distinguish between human users and automated bots on websites.

A comprehensive, interactive CAPTCHA simulator that demonstrates 4 different types of CAPTCHA verification methods used in real-world web applications.

## ✨ Features

- **4 CAPTCHA Types** implemented from scratch:
  - 📝 **Text-based CAPTCHA** - Distorted characters with visual noise
  - 🔊 **Audio-based CAPTCHA** - Speech synthesis audio verification (accessibility feature)
  - 🧮 **Mathematical Puzzle** - Simple arithmetic challenges
  - 🖼️ **Image-based Grid** - Object recognition in 3x3 grid

- **User-friendly Flow**:
  1. Enter your name
  2. Progress through all 4 CAPTCHA types
  3. Get verified with personalized welcome message
- **Visual Progress Tracker** - See your verification progress in real-time
- **Pure Frontend** - Built with HTML, CSS, and JavaScript - no backend required

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Web Speech API (for audio CAPTCHA)
- Vercel (Deployment)

## 📋 How It Works
1. **Step 1**: Enter your name and read about the 4 CAPTCHA types
2. **Step 2**: Solve the distorted text CAPTCHA
3. **Step 3**: Listen to and type the audio CAPTCHA
4. **Step 4**: Solve the math puzzle
5. **Step 5**: Select all images containing the target object
6. **Final**: Get your personalized "Verified! Welcome [name]" message
